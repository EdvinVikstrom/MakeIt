# MakeIt
Makefile generator

# Building
- Linux: `make`
- Windows: `make`

# Installing
- Linux: `make install`
- Windows: `make install`

# Running
Run the `makeit` executable to read the default target(`MIBuild`).
Run `makeit --help` for help.
